Welcome to Knife! (BETA)

What is Knife?

Knife is a collection of commands and scripts designed for use with the CentOS operating system.
I'm always trying to add more things, if you have a suggestion let me know.
The below commands are accessed from the Linux terminal using curl with the URLs specified below.
For example curl knife.so/phpinfo.

WordPress Version Checker - knife.so/wordpress

A short Bash script that will return the latest version of the WordPress CMS and show you the
versions installed on the server. Requires the mlocate package to work. Run this with curl knife.so/wordpress | sh.
Alternatively you can pop it into a file if you don't want to blindly run code on your server, which is sensible.

DDoS Deflate - knife.so/deflate

The same script that can be found at deflate.medialayer.com I just wanted it to be more easily accessible from the command line.
Returns the command found on the link above.

PHP Info page - knife.so/phpinfo

Returns the phpinfo() function in PHP tags because I can never remember the syntax.

Speedtest - knife.so/speedtest

A simple 500Mb speedtest that outputs to /dev/null.

Exim spam checker - knife.so/eximspam

Scans the exim_mainlog and returns spamming scripts

Postfix spam checker - knife.so/postfixspam

Scans /var/spool/postfix and returns spamming scripts

Show all crons - knife.so/allcron

Loops through users in /etc/passwd and lists their crontab
